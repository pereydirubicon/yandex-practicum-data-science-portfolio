# Проект: Прогнозирование наиболее прибыльного региона для бурения нефтяных скважин

## Описание проекта

Добывающей компании «ГлавРосГосНефть» необходимо определить наиболее перспективный регион для разработки новых нефтяных месторождений. У компании есть пробы нефти в трёх регионах с измеренными характеристиками скважин.

Бюджет на разработку скважин в каждом регионе — **10 миллиардов рублей**. При текущих ценах на нефть доход с каждой скважины составляет 450 тысяч рублей. Для безубыточности средний запас по региону должен превышать пороговое значение.

**Цель проекта:** построить модель машинного обучения, которая поможет выбрать регион с максимальной прибылью и минимальным риском убытков.
