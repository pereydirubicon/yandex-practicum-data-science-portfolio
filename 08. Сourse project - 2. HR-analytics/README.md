# Проект: Прогнозирование уровня удовлетворенности и риска увольнения сотрудников 👥

## Описание проекта

HR-отдел компании «Работа с заботой» поставил задачу: оптимизировать управление персоналом через прогнозирование ключевых HR-показателей с помощью машинного обучения.

Для этого необходимо построить две прогнозные модели:

1. Модель регрессии — для предсказания уровня удовлетворённости сотрудника работой в компании (`job_satisfaction_rate`)
2. Модель классификации — для предсказания риска увольнения сотрудника (`quit`)

**Цель проекта:** разработать модели, которые помогут HR-отделу выявлять сотрудников с низкой удовлетворённостью и высоким риском увольнения, а также предложить рекомендации для снижения текучести кадров.
