# Проект: Прогнозирование оттока клиентов интернет-магазина 🛒

## Описание проекта

Руководство интернет-магазина «В один клик» поставило задачу: разработать систему персонализированных предложений для постоянных клиентов, чтобы увеличить их покупательскую активность и снизить риск оттока.

Для этого необходимо построить прогнозную модель, которая будет предсказывать вероятность снижения покупательской активности клиента в следующие три месяца.

**Цель проекта:** разработать модель классификации для выделения сегмента клиентов с высоким риском оттока и предложить персонализированные рекомендации для их удержания.

Обнаружена высокая доля клиентов, согласных на коммуникации (74%).</li><li>**Объединение данных:** создание единой таблицы `modeling_df` (1296 строк, 18 столбцов) с отдельными столбцами для выручки и времени по каждому периоду. Добавлен новый признак `diff_revenue` (разница выручки).</li><li>**Корреляционный анализ:** построена матрица корреляций Спирмена. Выявлены сильные положительные связи целевого признака с `pages_per_visit` (0.55) и `previous_month_minutes` (0.52). Обнаружена сильная отрицательная связь с `promotional_purchases` (-0.47).</li><li>**Моделирование:** построен пайплайн предобработки (OneHotEncoder, OrdinalEncoder, MinMaxScaler). Лучший результат показала модель **SVC** с метрикой **ROC-AUC 0.91** на тестовой выборке.</li><li>**Интерпретация:** с помощью SHAP определены ключевые признаки, влияющие на отток: `promotional_purchases`, `pages_per_visit`, `previous_month_minutes`, `current_month_minutes`, `avg_categories_per_visit`, `unpaid_products_quarter`.</li><li>**Сегментация:** выделен целевой сегмент клиентов, у которых текущая активность стабильна, но прогнозируется снижение в следующем месяце. Чем выше вовлечённость, тем ниже риск оттока.</li><li>Высокая доля покупок по акциям (`promotional_purchases`) значимо коррелирует со снижением активности.
