# Проект: Прогнозирование стоимости жилья в Калифорнии

## Описание проекта

**Цель проекта:** Обучить модель линейной регрессии для предсказания медианной стоимости недвижимости в Калифорнии (данные 1990 года) с использованием библиотеки машинного обучения Apache Spark MLlib.

**Задачи:**
1. Инициализировать локальную Spark-сессию для обработки данных
2. Провести исследовательский анализ данных и подготовить их к моделированию
3. Обучить две модели линейной регрессии:
   - Модель №1 — с использованием всех признаков (включая категориальный)
   - Модель №2 — только с числовыми признаками
4. Оценить качество моделей на тестовой выборке с помощью метрик RMSE, MAE и R²
5. Сделать вывод о влиянии категориального признака на точность предсказаний
