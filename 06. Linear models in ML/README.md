# Проект: Прогнозирование удоя и вкуса молока 🐄

## Описание проекта

Фермер, владелец молочного хозяйства «Вольный луг», планирует расширить поголовье и закупить новых коров у стороннего поставщика («ЭкоФерма»). Чтобы минимизировать риски и не потерять в качестве продукции, ему необходимо отобрать только тех коров, которые:

- дают не менее **6000 кг молока в год**
- дают **вкусное молоко** (по его личной оценке)

**Цель проекта:** разработать две прогнозные модели для отбора бурёнок в поголовье:

1. Модель регрессии — для предсказания годового удоя (`milk_kg`)
2. Модель классификации — для расчёта вероятности получить вкусное молоко (`taste_milk`)

| **Ход проекта** | <ol><li>**Загрузка и изучение данных:** знакомство с тремя таблицами (`ferma_main`, `ferma_dad`, `cow_buy`), первичный анализ структуры и типов данных.</li><li>**Предобработка:** удаление дубликатов, замена неявных дубликатов, устранение аномалий (удалено значение 45 616 кг в столбце `milk_kg`).</li><li>**EDA:** визуализация распределений количественных и категориальных признаков, выявление особенностей (высокий процент нарушений, дисбаланс в повышениях, высокая текучка после 2 лет).</li><li>**Корреляционный анализ:** построение матрицы phik-корреляций, диаграмм рассеяния с линиями тренда. Выявлена сильная связь удоя с `eke` (0.72) и `fat_content_percent` (0.71), а также умеренная нелинейность для `spo` и `eke`.</li><li>**Моделирование (регрессия):** построены три модели линейной регрессии с последовательным улучшением: базовая (R² 0.75), с учётом нелинейности (R² 0.79), с добавлением новых признаков (R² 0.80).</li><li>**Моделирование (классификация):** построена модель логистической регрессии.
